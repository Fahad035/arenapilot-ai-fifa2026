# Frequently Asked Questions

---

## What is ArenaPilot AI?

ArenaPilot AI is a Generative AI-powered stadium operations platform designed to assist venue operators in managing crowd safety, incidents, and operational decision-making.

---

## Which AI model is used?

ArenaPilot AI uses Google Gemini to generate operational briefings, recommendations, and intelligent insights.

---

## Can the application work without AI?

Yes.

If the Gemini API is unavailable or quota limits are reached, the application automatically switches to realistic mock responses to ensure uninterrupted functionality.

---

## Which frontend framework is used?

- React
- Vite
- Tailwind CSS
- Framer Motion

---

## Which backend technology is used?

- Node.js
- Express.js

---

## Is the application responsive?

Yes.

ArenaPilot AI is fully responsive and optimized for desktop, tablet, and mobile devices.

---

## Does it support real-time monitoring?

The platform simulates live operational monitoring through dynamic dashboards, live alert tickers, and AI-generated updates.

---

## Can this be used for other events?

Yes.

The platform can be adapted for:

- Football Stadiums
- Cricket Stadiums
- Concert Venues
- Festivals
- Conferences
- Emergency Drills

---

## Is authentication implemented?

Not in the current version.

Future versions will include secure user authentication and role-based access control.
